# Nikhil Tiwari — Backend Engineer Portfolio

Cinematic hero + professional portfolio for resume, LinkedIn, and recruiters.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize your content

Edit **`lib/portfolio/profile.ts`** — name, email, LinkedIn, GitHub, skills, experience, and projects.

| Asset | Path |
|--------|------|
| Intro video | `public/Use_the_uploaded_image_as_the.mp4` |
| Hero still (after video) | `public/hero-poster.png` |
| Resume PDF | `public/resume.pdf` |

## Deploy (Vercel)

1. Push to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Set env: `NEXT_PUBLIC_SITE_URL=https://your-domain.com` (for Open Graph / LinkedIn previews)

## LinkedIn

Add your live URL to LinkedIn **Featured** or **Contact info → Website**.

## Build

```bash
npm run build
npm start
```
